# Dijital Tesis Platformu v1

AI destekli, veritabanı tabanlı, kullanıcı girişli dijital tesis proje yönetim platformu.

## Bu paket neyi çözer?

- Yeni HTML versiyonu üretmeden proje düzenleme.
- Her kullanıcı için ayrı proje listesi.
- Admin için tüm kullanıcı/proje/değişiklik görünürlüğü.
- Boş tesis projesiyle başlama.
- Bina, kolon, makine, raf, levha, yangın ekipmanı ekleme.
- AI komutunu güvenli işlem planına dönüştürme.
- Kullanıcı onayıyla proje verisine uygulama.

## Kurulum özeti

1. GitHub'da yeni repo aç: `dijital-tesis-platformu`
2. Bu paketin içindeki dosya/klasörleri repoya yükle.
3. Supabase'te yeni proje aç.
4. `database/001_SUPABASE_SETUP.sql` dosyasını SQL Editor'da çalıştır.
5. Vercel'de GitHub reposunu import et.
6. Environment Variables gir.
7. Deploy et.

Detaylı adımlar: `docs/KURULUM_ADIM_ADIM.md`
